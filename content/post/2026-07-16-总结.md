# 网络编程2

## 同步续

### 解析域名

```cpp
int dns_conect_to_end()
{
	std::string host = "example.com";//域名
	std::string port_num = "3333";//端口
	asio::io_context ios;//上下文
	asio::ip::tcp::resolver resolver(ios);//创建dns解析器对象
	boost::system::error_code ec;
	auto results = resolver.resolve(host, port_num, ec);//解析域名
	if (ec)
	{
		std::cout << "DNS 解析失败，错误码 = " << ec.value() << "，信息：" << ec.message() << std::endl;
		return ec.value();
	}
	return 0;
}
```
