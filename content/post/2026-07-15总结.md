# 网络编程1

## 使用模式

### proactor

- 自动读取
- 异步模型

### reactor

- epoll模式
- 同步模型

## 基本流程（传统模式）

- 采用阻塞模型

### 服务器

- 1.socket-创建socket对象
- 2.bind-绑定本机ip+port
- 3.listen-监听窗口，以用于建立连接
- 4.accept-创建一个socket对象发送消息(实际中因为是多连接的，所以需要创建多个socket对象去进行收发)
- 5.read、write-收发消息

### 客户端

- 1.socket-创建socket对象
- 2.connect-根据服务器的ip+port发起连接请求
- 3.write、read-建立连接收发消息

## 连接端口

```cpp
//客户端
int client_end_point()
{
	std::string raw_ip_address = "127.4.8.1";//目标地址
	unsigned short port_num = 3333;//目标端口
	boost::system::error_code ec;//错误对象
	asio::ip::address ip_address = asio::ip::make_address(raw_ip_address, ec);//将ip与端口绑定
	if (ec.value() != 0)//ec有错误进入
	{
		std::cout << "failed to parse the IP address. Error code = " << ec.value() << ".Message is" << ec.message();
		return ec.value();//返回错误码
	}
	asio::ip::tcp::endpoint ep(ip_address, port_num);//创建 TCP 端点 (endpoint) 连接对象
	return 0;
}

//服务端
int server_end_point()
{
	unsigned short port_num = 3333;//端口号
	asio::ip::address ip_address = asio::ip::address_v4::any();//监听哪些网卡的ip
	asio::ip::tcp::endpoint ep(ip_address, port_num);//创建一个 TCP 连接对象
	return 0;
}
```

## 同步写法

### 之前的写法

```cpp
//创建socket
int create_tcp_socket()
{
	asio::io_context ioc;//创建io上下文
	asio::ip::tcp protocol = asio::ip::tcp::v4();//指定tcp协议
	asio::ip::tcp::socket sock(ioc);//创建一个 未连接的 TCP socket，并将其与 io_context 关联
	boost::system::error_code ec;
	sock.open(protocol, ec);//显式打开一个 socket，并将其与指定的协议关联。
	if (ec.value())
	{
		std::cout << "failed to parse the IP address. Error code = " << ec.value() << ".Message is" << ec.message();
		return ec.value();
	}
	return 0;
}

//接受socket
int create_accept_socket()
{
	asio::io_context ios;//TCP的上下文
	asio::ip::tcp::acceptor acceptor(ios);//TCP 接受器 (acceptor) 对象，用于在服务器端监听和接受传入的连接请求。
	asio::ip::tcp protocol = asio::ip::tcp::v4();//指定tcp协议
	boost::system::error_code ec;
	acceptor.open(protocol, ec);
	if (ec) {
        std::cout << "bind failed: " << ec.message();
        return ec.value();
    }
	return 0;
}
```

##### 绑定acceptor

```cpp
int bind_acceptor_socket() {
	unsigned short port_num = 3333;//端口号
	asio::ip::tcp::endpoint ep(asio::ip::address_v4::any(),port_num);//指定所有的网卡为接收网卡
	asio::io_context ios;//上下文
	asio::ip::tcp::acceptor acceptor(ios, ep.protocol());//TCP接收器
	boost::system::error_code ec;
	acceptor.bind(ep, ec);//绑定
	acceptor.listen(asio::socket_base::max_listen_connections, ec);//监听
    if (ec) {
        std::cout << "listen failed: " << ec.message();
        return ec.value();
    }
	return 0;
}
```

### 连接服务器
```cpp
int connect_to_end() 
{
	std::string raw_ip_address = "192.168.0.102";//ip
	unsigned short port_num = 3333;//端口
	try
	{
		asio::ip::tcp::endpoint ep(asio::ip::make_address(raw_ip_address), port_num);//创建端口
		asio::io_context ios;//TCP上下文
		asio::ip::tcp::socket sock(ios, ep.protocol());//创建socket
		sock.connect(ep);//连接
	}
	catch (system::system_error& e)
	{
	}
}
```
