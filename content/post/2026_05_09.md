# web && http
## http
### http连接
#### 非持久HTTP
- 最多只有一个对象在TCP连接上发送
- 下载多个对象需要多个TCP连接
- HTTP/1.0开始使用非持久连接
#### 持久HTTP
- 多个对象的TCP连接传输
- HTTP/1.1默认使用持久连接
### 请求报文
#### 表单输入
##### Post方式
- 网页通常包括表单输入
- 包含在实体中的输入提交到服务器
##### URL
- GET方式提交
- 输入通过请求行的URL字段上载
#### 获取方式
##### HTTP/1.0
- GET
- POST
- HEAD
##### HTTP/1.1
- GET,POST,HEAD
- PUT(提交)
- DELETE(删除)

#### 结构
##### 状态行
